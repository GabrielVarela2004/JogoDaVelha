# Jogo da Velha (Tic-Tac-Toe) em Java ☕🎮

Jogo da Velha clássico desenvolvido para rodar diretamente no terminal, criado como um projeto de estudo para consolidar fundamentos de lógica de programação e a base do ecossistema Java.

## 🛠️ O que foi praticado neste projeto:
- **Estruturas de Dados:** Uso de matrizes (`char[][]`) para modelar e renderizar o tabuleiro 3x3.
- **Controle de Fluxo e Loops:** Alternância dinâmica de turnos entre jogadores (`boolean`) e loops de validação.
- **Modularização:** Separação da lógica de verificação de vitória (linhas, colunas e diagonais) em métodos dedicados.
- **Tratamento de Entradas:** Tratamento de regras de negócio para evitar coordenadas repetidas ou fora do limite do tabuleiro (1 a 3).

## 🚀 Como rodar o projeto:
1. Certifique-se de ter o JDK instalado.
2. Abra o projeto em sua IDE de preferência (desenvolvido utilizando IntelliJ IDEA).
3. Execute a classe principal `Tabuleiro.java`.
